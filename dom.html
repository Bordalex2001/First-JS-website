Document Object Model (DOM)

DOM — концепція, згідно з якої всі елементи відображаються на об'єкті (ООП) JS. Причому це відображення двобічне: зміна HTML відбивається на структурі об'єктів, а зміна структури об'єктів — на HTML. Іншими словами, немає різниці як маніпулювати сторінкою — через вплив на HTML чи шляхом зміни об'єктів (та їх колекцій).
