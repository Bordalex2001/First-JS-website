Полотно. Засоби малювання

Основу для створення складних зображень дає елемент <canvas> — полотно. З урахуванням особливості DOM — перезбирати весь документ після довільної зміни — полотна надають покращені можливості створення динамічного контенту.
Лінії. Ідея малювання ліній полягає у комбінації двох операцій — переміщення "каретки" без сліду, та переміщення зі слідом.
За замовчанням розмір полотна 300 х 150 пікселів. При цьому важливо розуміти, що є розміри, а є роздільна здатність. Якщо стилями змінити ромір полотна, то його роздільна здатність не зміниться, при збільшенні проявиться піксельна структура.

Кешування (збереження) даних

  • Зменшення трафіку
  • Автономність (можливість роботи без Інтернету)
  • Збереження налаштувань

У JavaScript у бразері є ряд обмежень, зокрема, доступ до файлової системи. Для задач збереженння даних браузер надає декілька засобів, доступних для перегляду у вкладці "Application" інструментів розробника (консолі):

  • Local Storage — постійне сховище. Тримає значення, що доступне всім вкладкам, в яких відкрито дану сторінку. Відновлює значення після закриття вкладки і навіть браузера.
  • Session Storage — тимчасове сховище. Значення доступне лише даній вкладці. Закриття вкладки руйнує занчення. При оновленні сторінки значення зберігається.
  • Indexed DB — для великих даних типу "ключ-значення"
